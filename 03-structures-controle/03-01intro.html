JavaScript -III/01 -Structures de contrôle - Le constantes...

IL existe 2 types de Structures de contrôle de base , que l'on retrouve dans tous les langages de programmation , et donc en JS

Les conditions & les boucles

les conditions vont nous permettre de d'éxécuter un certain nombre d'instruction si et seulment si une certaine condition est vérefiée

ex: Si il pleut je ne sors pas

les boucles

Les boucles vont nous permettre d'éxécuter un bloc des condition en boucle tant qu'une condition dinnée est vérifiée

ex: Dés que j'ai fait monter les 12 éléves de ma class dans le bus ; je pars.

C-opérateurs de comparaison

comment vérifier les condition ?

Opérateurs de comparaison
Opérateur Définition

==

===

!=

<> (à éviter)

Teste l'égalité sur les valeurs

Teste l'égalité en terme de valeurs ET de types

Teste la différence des valeurs

Teste (également) la différences des valeurs

!==

Teste la différence en valeurs OU en types

<

>

<=

>=

Teste si une valeur est strictement inférieure à une autre

Teste si une valeur est strictement supérieure à une autre

Test si une valeur est inférieure OU égale à une autre

Test si une valeur est supérieure OU égale à une autre

Dans le cas où la comparaison est vérifié, le JS renvoie le booléen true, dans le cas contraire, le booléen false est renvoyé.

Le JS va automatiquement comparer la valeur à gauche de l'opérateur à celle de droite selon l'opérateur de comparaison.
