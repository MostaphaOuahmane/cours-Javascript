le Java-script est un langage dynamique

Ce langage va nous permettre de manipuler des contenus HTML ou des styles CSS et de les modifier en fonction de divers évènements ou variables. Un évènement peut être par exemple un clic d'un utilisateur à un certain endroit de la page tandis qu'une variable peut être l'heure de la journée

le Java-script est un langage principalement côté client

JS s'éxécute côté client, nous allons quand même pouvoir l'utiliser côté serveur à condition que l'on mette en place un environnement favorable, en utilisant, par exemple node.js.

Java-Script est un langage interprété

Pour être interprété js doit être éxécute directement dans un logiciel interprété

JavaScript n'est pas JAVA

Encore aujourd’hui, certaines personnes ont tendance à confondre les deux langages « Java » et « JavaScript ». Retenez ici que ces deux langages, bien que syntaxiquement assez proches à la base, reposent sur des concepts fondamentaux complètement différents et servent à effectuer des tâches totalement différentes.
